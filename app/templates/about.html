About

切り抜き検索くん について

元動画の URL を貼るだけで、関連する切り抜き動画を探せるシンプルな検索サービスです。 動画ファイルの保存や再配信は行わず、視聴はすべて YouTube 上で完結します。

仕組み

入力された元動画の URL または動画 ID をもとに、YouTube Data API を使って 公開されている動画のタイトルや説明文を検索し、元動画 URL を含む動画を 「切り抜き候補」として表示します。

再生数・投稿日・動画の長さなどで並び替えて確認でき、 気になった動画にはお気に入りを付けてまとめておくこともできます。

About this service (English)

切り抜き検索くん lets you find “clip videos” based on an original YouTube live stream or VOD URL. We query public video metadata via the YouTube Data API to surface videos that mention the original URL or video ID.

We do not download or re-host any video content. All videos are watched directly on YouTube.
