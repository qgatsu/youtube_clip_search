Privacy

プライバシーポリシー

切り抜き検索くん で取り扱う情報と、その利用目的についてまとめています。 重要なポイントを簡潔に記載していますのでご確認ください。

1. 取得する情報

本サービスは、以下の情報を取得・利用する場合があります。

  • ユーザーが入力した元動画の URL およびそこから取得される動画 ID
  • YouTube Data API Services を通じて取得する公開動画のメタデータ（タイトル、チャンネル名、公開日時、サムネイル等）
  • 元動画と切り抜き候補動画の動画 ID、検索日時などのログ情報
  • アクセスログ（IP アドレス、ブラウザ情報等）

2. 利用目的

取得した情報は、以下の目的で利用します。

  • 元動画に関連する切り抜き動画を検索・表示するため
  • 本サービスの品質向上・不具合対応等のための統計的な分析のため

3. 保存期間

動画 ID や検索日時等のログは、サービス運営および分析のために合理的な期間（目安として3ヶ月程度）保存し、その後削除または匿名化します。 必要に応じてこの期間は変更される場合があります。

4. YouTube / Google に関する事項

本サービスは、YouTube Data API Services を利用して YouTube 上の公開データのみを取得します。 本サービスの利用には YouTube 利用規約 および Google プライバシーポリシー が適用されます。

本サービスは、ユーザーの Google アカウント情報や非公開データにはアクセスしません。
